config

Default configuration for pulse-sequence plotting.

DEFAULT_CHANNEL_ORDER module-attribute ¶

DEFAULT_CHANNEL_ORDER = ['MW', 'EOM', 'PMT', 'AOM']

Order in which channels are drawn (top→bottom).

DEFAULT_COLORS module-attribute ¶

DEFAULT_COLORS = {'MW': '#B0B0B0', 'EOM': '#A0D468', 'PMT': '#4C4C4C', 'AOM': '#4A89DC'}

Default fill colors for each channel.

RC_PARAMS module-attribute ¶

RC_PARAMS = {'font.size': 16, 'axes.titlesize': 18, 'axes.labelsize': 16, 'xtick.labelsize': 16, 'ytick.labelsize': 16}

Global Matplotlib rcParams applied to pulse plots.

BASELINE_HEIGHT module-attribute ¶

BASELINE_HEIGHT = 0.4

Relative height of each pulse bar.

SEPARATOR_STYLE module-attribute ¶

SEPARATOR_STYLE = {'linestyle': '--', 'linewidth': 0.8, 'color': 'grey'}

Style for dashed separators between time sections.

TIME_AXIS_PROPS module-attribute ¶

TIME_AXIS_PROPS = {'arrowstyle': '->', 'lw': 1.5, 'color': 'black'}

Style for the time-axis arrow.

FONT_SIZE module-attribute ¶

FONT_SIZE = 12

Font size for pulse labels.

FONT_FAMILY module-attribute ¶

FONT_FAMILY = 'Cambria'

Font family for pulse labels.

FONT_COLOR module-attribute ¶

FONT_COLOR = 'white'

Text color for pulse labels.

HORIZONTAL_SHIFT module-attribute ¶

HORIZONTAL_SHIFT = 0.0

x-offset applied to label position (in time units).

VERTICAL_SHIFT module-attribute ¶

VERTICAL_SHIFT = 0.0

y-offset applied to label position (axes units).
